# 바위게마을 Online Beta 1.3.7.15

## 게시글 즐겨찾기
- 게시글 상세 화면에서 ☆ 버튼으로 즐겨찾기 추가
- 즐겨찾기된 글은 ★로 표시
- 게시판 상단의 `★ 내 즐겨찾기`에서 저장한 글 확인
- 최근 저장한 순서로 정렬
- 운영진 전용 등 접근할 수 없는 게시글은 목록에서 자동 제외
- 게시글 삭제 시 연결된 즐겨찾기도 자동 삭제
