# 바위게마을 Online Beta 1.3.6.1

## 변경
- 왼쪽 사이드바를 네이버 카페형 대분류/소분류 구조로 변경
- 게시판 대분류를 접기/펼치기 가능
- 소분류를 누르면 해당 게시판으로 바로 이동
- 게시판 내부의 큰 대분류/소분류 탭 제거
- 모바일에서는 게시판 선택 드롭다운 제공

## 운영진 설정
- 대분류 추가·수정·삭제
- 소분류 추가·수정·삭제
- 이름, 아이콘, 순서 설정
- 메뉴 표시/숨김 설정
- 클랜원 공개/운영진 전용 설정
- 설정 저장 즉시 왼쪽 사이드바에 자동 반영
