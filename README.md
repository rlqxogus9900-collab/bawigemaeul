## 홈페이지 속도 개선
- 공지는 즉시 반영을 유지하면서 별도 조회
- 규칙·최근 내전·클랜원 라인·후원 목록은 2분 서버 캐시 적용
- 홈 접속 때마다 전체 클랜원 상세정보를 읽지 않고 `main_line`만 조회
- 최근 내전 결과도 홈에서 사용하는 필드만 조회
- 데이터 집계는 한 번만 수행해 캐시 결과 재사용
- 홈 링크의 과도한 자동 프리페치 비활성화

- 공지 등록·수정·삭제는 기존처럼 홈에 즉시 반영
