# 바위게마을 Online Beta 1.2

## 이번 버전 추가
- 운영진이 클랜원 홈페이지 닉네임 수정
- Riot ID 수정
- 클랜원/운영진 권한 변경
- 계정 활성/비활성 변경
- 중복 닉네임 및 Riot ID 차단
- 본인 운영진 계정의 권한 회수/비활성화/삭제 방지
- 비밀번호 1234 초기화와 계정 삭제 유지

## 업데이트 방법
1. ZIP 압축 해제
2. 폴더 안의 파일과 폴더 전체를 GitHub 저장소 최상위에 업로드
3. Commit message 예: `Update Online Beta 1.2`
4. Vercel 자동 배포 완료 대기

## 닉네임 수정 후
- 수정된 계정은 다음 로그인부터 새 홈페이지 닉네임을 사용합니다.
- 이미 로그인 중인 사용자는 로그아웃 후 새 닉네임으로 로그인하면 됩니다.
