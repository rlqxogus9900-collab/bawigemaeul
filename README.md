# 바위게마을 Online Beta 1.3.8.0

## 실제 추가 기능
- 실시간 경매 1차 구현
- 연동 정기내전 투표에서 참가자/팀장 자동 불러오기
- 팀장은 경매 선수 명단에서 자동 제외
- 경매방 생성 및 시작
- 선수 지명, 팀별 입찰, 낙찰, 유찰, 종료
- 팀별 남은 예산 및 낙찰 선수 실시간 표시
- 입찰 기록 표시
- 모든 접속 화면 2초 간격 자동 동기화
- 운영진만 경매 조작 가능, 일반 클랜원은 관전 가능

## 배포 전 필수
`SQL-실행용-1.3.8.0.sql` 전체를 Supabase SQL Editor에서 한 번 실행하세요.
