# 바위게마을 Online Beta 1.3.8.1

## 실제 추가 기능
- 지정된 팀장이 로그인 후 본인 팀으로 직접 입찰
- 운영진은 모든 팀 입찰 및 경매 진행 가능
- 방송 전용 페이지 `/auction/broadcast`
- 팀 명단 좌우·다중 자동 배치
- 현재 입찰 팀 빛나는 강조
- 낙찰 시 전체화면 SOLD 애니메이션
- 입찰 효과음 ON/OFF
- 세이프 에어리어 ON/OFF
- 방송 UI 숨기기 및 전체화면 버튼
- 업데이트 페이지에 1.3.8.0 / 1.3.8.1 최신 내역 추가
- 업데이트 페이지 1시간 캐시 제거

## SQL
추가 SQL 없음

## 사용
1. 운영진이 실시간 경매 페이지에서 경매방을 생성하고 시작합니다.
2. 투표에서 지정된 팀장이 자기 계정으로 로그인합니다.
3. 팀장에게는 자기 팀 입찰 버튼만 활성화됩니다.
4. 방송 화면 열기 버튼을 눌러 디스코드 화면 공유 또는 OBS 브라우저 화면으로 사용합니다.
