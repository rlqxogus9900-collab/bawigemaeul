# 바위게마을 Online Beta 1.3.7.11

## 게시판 관리
- 대분류와 소분류의 숫자 순서 입력칸 제거
- ▲ 버튼으로 한 칸 위 이동
- ▼ 버튼으로 한 칸 아래 이동
- 첫 항목의 ▲, 마지막 항목의 ▼ 자동 비활성화
- 새 게시판은 해당 목록의 맨 아래에 자동 추가
- 변경 순서가 왼쪽 메뉴와 게시판 목록에 즉시 반영

## 홈 최근 공지 수정
- 홈 공지 캐시 제거
- 공지 등록·수정·삭제 후 홈 화면 즉시 갱신
- 고정 여부와 관계없이 실제 작성일 기준 최신 공지 5개 표시
- 전체 공지에는 보이지만 홈에는 안 보이던 문제 수정
