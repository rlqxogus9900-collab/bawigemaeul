# 바위게마을 Online Beta 1.3.7.19

## 알림 시스템
- 로그인 사용자 상단에 알림 종 아이콘 추가
- 안 읽은 알림 개수 배지 표시
- 최근 알림 20개 표시
- 새 공지 등록 시 전체 활성 클랜원에게 알림
- 새 일반/정기내전 투표 등록 시 전체 활성 클랜원에게 알림
- 내 게시글에 다른 사용자가 댓글 작성 시 알림
- 알림 클릭 시 관련 페이지 이동 및 읽음 처리
- 전체 읽음 기능
- 1분마다 새 알림 자동 확인
- 모바일 알림 팝업 지원
