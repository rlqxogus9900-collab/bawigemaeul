# 바위게마을 Online Beta 1.3.8.2

## 실제 반영
- 경매방 생성 위치를 `경매 관리`로 이동
- 경매 관리에서 방을 만들면 실시간 경매와 방송 화면에 자동 표시
- 실시간 경매 페이지에서는 더 이상 방을 만들지 않음
- 입찰 금액 상승 애니메이션 추가
- 현재 최고 입찰 팀 반짝임 효과 추가
- 낙찰 시 실시간 경매 화면에도 전체화면 SOLD 애니메이션 추가
- 경매 종료 후 팀별 팀장, 낙찰 선수, 선수별 낙찰 점수, 잔여 예산 표시
- 종료된 최신 경매 결과가 새 방 생성 전까지 유지
- 입찰 기록 글자색 수정 및 입찰 팀·금액·입찰자·시간 표시
- 방송 화면 입찰 기록 글자색 수정
- 홈 화면 최신 업데이트 카드 1.3.8.2로 갱신
- 업데이트 페이지 1.3.8.2 누적 추가
